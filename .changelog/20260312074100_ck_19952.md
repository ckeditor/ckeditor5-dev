Added a `disallowLatestNpmTag` option to `publishPackages()` and npm tag validation so release workflows can explicitly block publishing with the `latest` [dist-tag](https://docs.npmjs.com/cli/commands/npm-dist-tag).

When enabled, attempting to publish with `npmTag: 'latest'` now fails before publishing starts.
